# 10_sequences.md — BB2025 sequence generators (Java → Rust)

Base: `ffb-server/.../step/generator`. **BB2025 has its own `@RulesCollection(BB2025)`
subclass for every generator in scope** (`generator/bb2025/`), each extending the COMMON
abstract base in `generator/*.java` (which only declares `SequenceParams`). The bb2016/bb2020
siblings are NOT used for BB2025. Resolution: `SequenceGeneratorFactory.forName(Type.X)`.

`Sequence` helper: `add(StepId,[label],params…)`; `jump(label)` = `add(GOTO_LABEL, GOTO_LABEL=label)`;
`insertHooks(HookPoint,…)` injects skill-behaviour hook steps (none for skill-less linemen).
`GOTO_LABEL_ON_*` on a step = where it jumps on that outcome (matched to a step `add`ed with
that `IStepLabel`).
